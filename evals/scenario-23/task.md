# Paid Ads Budget Plan for PawBox Launch

## Problem/Feature Description

PawBox is a monthly subscription box for dog owners, launching in six weeks. The founders have never run paid social advertising before and are asking for help structuring their first paid media budget. They have a $5,000/month total budget to work with across Meta (Facebook/Instagram) only, and their product catalog currently has three subscription tiers: Pup Pack ($29/mo), Fetch Box ($49/mo), and Alpha Bundle ($79/mo).

The marketing lead wants a clear framework showing how to allocate the budget across different campaign objectives and audience types, with guidance on what changes as the brand matures from pre-launch through to scaling. The company has no existing customer list, no pixel data, and no prior ad performance to work from.

## Output Specification

Produce a markdown file named `ads-plan.md` in the current working directory that:

- Shows how the $5,000/month budget should be split across campaign types (cold, retargeting, awareness, catalog), with the exact percentages and dollar amounts
- Shows how those splits should change across the brand's lifecycle as it grows from launch through to maturity, with explicit percentage breakdowns for each stage
- Includes recommended daily budgets for each ad set type given that this is their first time running ads
- Describes the mix for the first month of live ads specifically

The document should be immediately usable by a media buyer to set up the first campaigns.
