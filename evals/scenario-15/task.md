# Full Homepage Build for PetBox Monthly

## Problem/Feature Description

PetBox Monthly is a subscription box for dog owners — every month subscribers receive a curated selection of toys, treats, chews, and accessories tailored to their dog's size and preferences. The Shopify store runs Horizon and the team wants a complete homepage template that takes visitors from an emotional "this is for your pup" hook all the way through to conversion and email capture for fence-sitters.

The product manager has specified that the page must include every section the business needs: a hero banner, a how-it-works walkthrough, a tier picker (three size plans: Small Pup $35/mo, Medium Mutt $49/mo, Big Dog $65/mo), a recent boxes gallery, customer testimonials, a FAQ, and an email signup section for visitors who aren't ready to subscribe.

The theme developer will push this directly to the Horizon staging store — the template file needs to be ready to go with no manual fixes. The previous attempt at adding an email signup section caused a Horizon schema error when pushed, so the developer needs to pick the right section type.

## Output Specification

Produce `templates/index.json` in the current working directory. Include all the sections described above. The email signup must function correctly as part of the homepage template. Make up realistic, fun copy for a dog subscription brand.
