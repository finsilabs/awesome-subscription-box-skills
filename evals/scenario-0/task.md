# Brand Identity Documentation for FabFitFun

## Problem/Feature Description

Your company has just signed FabFitFun as a client for a multi-channel marketing campaign. Before the creative team can produce any copy, design, or campaigns, everyone needs to work from the same brand reference — a centralized set of brand documentation files that captures their identity, tone, visual style, and positioning.

The marketing director has asked you to research FabFitFun's brand thoroughly by visiting their website (https://www.fabfitfun.com) and producing a complete, structured set of brand documentation files. These files should cover their verbal identity (tone, vocabulary, sample copy), visual identity (colors with their exact hex codes and roles, typography, photography direction), brand principles, and audience profile.

The goal is a self-contained, referenceable set of files that the entire team can use as a single source of truth for everything they write or design for this brand.

## Output Specification

Produce a set of brand documentation files organized into a single branded folder. The folder name should be derived from the brand name itself (use lowercase, hyphen-separated form). The files should be named and formatted so different team members (copywriters, designers, strategists) can each find the information relevant to their work.

At minimum, the documentation should cover:
- Voice and tone (including vocabulary guidelines and sample copy)
- Visual identity (colors with precise hex codes and their intended usage roles, typography, photo style)
- Brand principles and positioning
- Audience profile
- Sample quotes pulled directly from the website (with their source pages noted)
- A metadata file recording which pages were analyzed and when

All output files should be created and accessible so the team can review them.
